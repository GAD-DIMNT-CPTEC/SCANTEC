SCAMTEC INICIALIZAÇÃO
Data: 03/03/2015

Mini tutorial para instalação e execução testcase do SCAMTEC

1º Logar no supercomputador tupa 
---------------------------------------------------------------------------------------------------
2º Carregar os modulos do tupa com o comando
source /usr/bin/development_config

Escolher a opcao 6
---------------------------------------------------------------------------------------------------
3º Carregue o modulo pgi com o comando:
module load pgi
---------------------------------------------------------------------------------------------------
4º Entrar no diretorio escolhido para a instalação do SCAMTEC
Obs. Pode ser no $HOME/ $SUBMIT_HOME/ online6/ e outros
---------------------------------------------------------------------------------------------------
5º Baixar uma versao do SCAMTEC do repositorio svn (projetos.cptec.inpe.br)
svn export https://svn.cptec.inpe.br/scamtec/branches/lucamarante/SCAMTEC_v0.1@137
---------------------------------------------------------------------------------------------------
6º Entrar no diretorio do SCAMTEC
cd SCAMTEC_v0.1/
---------------------------------------------------------------------------------------------------
7º Execute o script install para compilar o sistema com o comando:
./install
Escolha a opção 2 para o compilador pgf do tupa

**(apenas para desenvolvimento de novo codigo):
Caso há mudança no codigo exporte a variavel do pgi_cray que esta no script install.
---------------------------------------------------------------------------------------------------
8º Entre no sequinte diretorio core onde estão os códigos do Scamtec.
cd core
---------------------------------------------------------------------------------------------------
9º Atualize o arquivo de configuração de acordo com o experimento:
gedit scamtec.conf&

-Configurar a data inicial e final, tempo de previsão e tempo de analise e total de previsao
-Configurar o recorte a ser escolhido (global, america sul ou brasil)
-Configurar o experimento referencia e experimentos. Adicionar o caminho abaixo nos dois experimentos para comparar ele versus ele mesmo:
/stornext/online6/assim_dados/lucas.amarante/TesteCase_SCAMTEC

-Configurar uso de climatologia e precipitacao (normalmente deixar o padrao que ja esta)
-Configurar o diretorio da saida dos dados
---------------------------------------------------------------------------------------------------
10º Execute o scamtec
./scamtec.x
